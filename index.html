🚀 Cosmos Cloud Marketplace

Personal Cosmos Cloud servapps marketplace by JesseRigon

📦 Add to Cosmos Cloud

In your Cosmos Cloud instance:

  1. Go to Settings → App Store
  2. Click Add Repository
  3. Paste this URL:
https://jesserigon.github.io/cosmos-marketplace/servapps.json

📊 Available Files

  • servapps.json - Main marketplace index (203 apps)
  • index.json - Extended marketplace data
  • servapps/ - Individual app definitions
